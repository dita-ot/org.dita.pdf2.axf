# AXF PDF plug-in

DITA-OT PDF plug-in for AntennaHouse Formatter.

## Compatibility

- DITA-OT 3.6

## Installation

Standard DITA-OT plug-in installation,
see [DITA-OT documentation](https://www.dita-ot.org/dev/topics/plugins-installing.html).

```shell
$ dita install org.dita.pdf2.axf
```
